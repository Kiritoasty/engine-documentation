Extra Characters

Adding extra characters, also should be fairly easy.

Step One: Character Duplication

Step Two: Character Values

You'll secondly want to duplicate the sprite named V5 (recommended) and then rename it to something like S1. But you can name it anything else. This stands for "Special 1"

Now that you have your sprite ready, change the "Assigned Character" value to "21".

Step Three: Stage Value Tinkering

Change the repeat value inside of the stage sprite that repeats 20, "add characters to 0", to "21".

Step Four: Icons

Change the repeat 10 value inside of the last part of the icons to 11 and make a new costumenamed your new extra character in the costume tabs with the assigned character value it is, an example is "21-a, 21-b, 21-c", a = idle, b = dragged, c = used

Alternatively, inside of everything of the text on top of us, you can also make it go to a new position, and set it to the new icon starting position and the amount of extra characters you have, you can also do more positions by doing this more and more, just not the same amount of extra characters you have.
